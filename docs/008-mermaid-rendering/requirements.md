# 008: Mermaid Diagram Rendering - Requirements

## Problem Statement

現在、コードブロックのシンタックスハイライト機能が実装されているが、`mermaid` 言語が指定されたコードブロックはMermaidダイアグラムとしてレンダリングされず、プレーンテキストまたはハイライトされたコードとして表示される。ドキュメント内のMermaid記法のダイアグラムを視覚的に確認できるようにする必要がある。

## Requirements

1. `` ```mermaid `` で記述されたコードブロックをMermaidダイアグラムとしてレンダリングする
2. Mermaid.jsライブラリをCDN経由でクライアントサイドに読み込み、ブラウザ上でレンダリングする
3. 通常のコードブロック（JavaScript、Python等）のシンタックスハイライトに影響を与えない
4. Mermaidダイアグラム以外のコードブロックは従来通りのシンタックスハイライトを適用する

## Constraints

- サーバーサイドレンダリングは行わず、クライアントサイドでMermaid.jsによりレンダリングする（Puppeteer等の重い依存を避けるため）
- Mermaid.jsはCDN経由で読み込み、npm依存として追加しない
- 既存のシンタックスハイライト機能を壊さない

## Acceptance Criteria

1. `` ```mermaid `` で記述されたコードブロックがSVGダイアグラムとして表示される
2. 通常のコードブロック（`javascript`, `python` 等）は従来通りシンタックスハイライトされる
3. 言語指定なしのコードブロックは従来通り自動検出でハイライトされる
4. 不明な言語のコードブロックは従来通りプレーンテキストとして表示される
5. 全ての既存テストが引き続きパスする

## User Stories

1. ユーザーとして、Markdownファイル内のMermaidダイアグラムをブラウザで視覚的に確認したい
2. ユーザーとして、Mermaidダイアグラムと通常のコードブロックが同じページに混在しても正しく表示されることを期待する
